# Gun Deaths in the US — Data Analysis & Clustering

Statistical analysis of US gun death data combining EDA with agglomerative clustering to find patterns across demographics, causes, and geography.

## Files
| File | Description |
|---|---|
| `Gun_Deaths.ipynb` | EDA — trends by year, age, race, intent |
| `AgglomerativeClustering.ipynb` | Hierarchical clustering on demographic profiles |
| `data/gun_deaths.csv` | Source dataset |

## Tech Stack
Python · pandas · numpy · matplotlib · seaborn · scikit-learn
