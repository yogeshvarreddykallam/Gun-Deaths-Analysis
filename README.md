# Gun Deaths in the US — Data Analysis & Clustering

Statistical analysis of US gun-related death data, combining exploratory data analysis with unsupervised machine learning (agglomerative clustering) to identify patterns across demographics, causes, and geography.

## Overview
- Exploratory Data Analysis: trends by year, age, race, intent, place
- Statistical visualizations using matplotlib and seaborn
- Agglomerative hierarchical clustering to find demographic groupings
- Dendrogram and cluster interpretation

## Files
| File | Description |
|---|---|
| `Gun_Deaths.ipynb` | EDA — trends, distributions, breakdowns |
| `AgglomerativeClustering.ipynb` | Hierarchical clustering on gun death profiles |
| `data/gun_deaths.csv` | Dataset (FiveThirtyEight gun deaths data) |

## Tech Stack
Python · pandas · numpy · matplotlib · seaborn · scikit-learn
